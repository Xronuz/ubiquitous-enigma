## 2. Architecture

```
School (tenant root)
  └── Users [9 roles: super_admin, school_admin, vice_principal, teacher,
              class_teacher, accountant, librarian, student, parent]
       └── Feature modules gated by SubscriptionPlan (free/basic/standard/premium)
              + ModuleName enum (19 modules toggled per school)
```

**Request flow:**
```
Client → nginx → Next.js (SSR/CSR) → Axios (auto-refresh JWT)
→ NestJS API v1 → TenantMiddleware → JwtAuthGuard → RolesGuard
→ SubscriptionGuard → Controller → Service → PrismaService (PG)
→ TransformInterceptor → { success, data, timestamp, path }
```

**Async flow:** Socket.IO (real-time) · BullMQ queues (notifications) · NestJS Schedule (cron: token cleanup, reminders)

**Multi-tenancy:** All DB models carry `schoolId` FK. TenantMiddleware extracts it from JWT.
